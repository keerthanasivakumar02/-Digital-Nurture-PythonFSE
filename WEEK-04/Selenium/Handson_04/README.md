# Hands-On 4 - Selenium WebDriver Basics

## Objective

This hands-on demonstrates the basic concepts of Selenium WebDriver using Python.

Topics Covered:
- Selenium Architecture
- WebDriver Setup
- Implicit Wait
- Headless Mode
- Browser Navigation
- Multiple Tabs
- Screenshot Capture
- Browser Window Size

- Python 3.12 or above

- Selenium
- webdriver-manager

Install the required packages:

```bash
pip install selenium webdriver-manager
```
