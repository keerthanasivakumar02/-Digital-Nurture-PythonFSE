# Hands-On 5 – Locators & Explicit Waits

## Objective
To learn different Selenium locator strategies and synchronization techniques using waits.

## Topics Covered
- ID Locator
- Name Locator
- Class Name Locator
- Tag Name Locator
- XPath
- CSS Selector
- WebDriverWait
- Expected Conditions
- Fluent Wait
- time.sleep() comparison

## Files

- task1_locators.py
  - Demonstrates different locator strategies.

- task2_waits.py
  - Uses WebDriverWait and Expected Conditions.

- task2_sleep.py
  - Demonstrates fixed wait using `time.sleep()`.

- task2_fluentwait.py
  - Demonstrates Fluent Wait using WebDriverWait with polling.

## Learning Outcome

- Understood different locator strategies.
- Learned to use explicit waits for reliable automation.
- Compared `time.sleep()` with `WebDriverWait`.
- Implemented Fluent Wait for dynamic elements.
