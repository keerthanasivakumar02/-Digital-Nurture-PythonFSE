# Hands-On 1: HTML5 Semantic Structure & CSS3 Foundations

## Student Portal

This project is developed as part of the **Digital Nurture 5.0 - Frontend Development** Hands-On 1.

## Objective

The objective of this hands-on is to build a Student Portal webpage using semantic HTML5 elements and CSS3 fundamentals. The project demonstrates proper webpage structure, styling, and responsive layout basics.

## Technologies Used

- HTML5
- CSS3
- Visual Studio Code
- Live Server Extension

## Project Structure

```
handson_01/
│
├── index.html
├── styles.css

```

## Features

- Semantic HTML5 structure
- Header with navigation menu
- Hero section with welcome message
- Explore Courses button
- Course cards
- Footer with developer information
- CSS Reset
- Flexbox Header Layout
- Button Hover Effect
- Box Shadow and Border Radius
