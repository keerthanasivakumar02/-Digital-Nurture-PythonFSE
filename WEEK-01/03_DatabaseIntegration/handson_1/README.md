# Hands-On 1: Database Schema Design & Core SQL

## Module
Digital Nurture 5.0 – Module 3: Database Integration

## Objective
This hands-on focuses on designing a relational database schema for a Student Course Registration System using MySQL. It covers database creation, table design, constraints, normalization, and schema modification using DDL commands.

## Topics Covered
- Database Creation
- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- UNIQUE Constraint
- NOT NULL Constraint
- CHECK Constraint
- ALTER TABLE
- Normalization (1NF, 2NF, 3NF)
