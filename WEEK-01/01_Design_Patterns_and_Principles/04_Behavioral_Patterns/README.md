# Behavioral Design Patterns

## Description

This folder contains simple Python programs that demonstrate the implementation of Behavioral Design Patterns. These examples illustrate how objects communicate, share responsibilities, and work together to perform different operations.

## Files Included

| File | Description |
|------|-------------|
| `02_Observer.py` | Demonstrates the Observer Pattern |
| `03_Strategy.py` | Demonstrates the Strategy Pattern |
| `04_Command.py` | Demonstrates the Command Pattern |

## Technologies Used

- Python 3.x
- Visual Studio Code
