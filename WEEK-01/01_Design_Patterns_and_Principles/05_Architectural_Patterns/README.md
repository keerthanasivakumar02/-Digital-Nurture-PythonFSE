# Architectural Design Patterns

## Description

This folder contains simple Python examples demonstrating commonly used Architectural Design Patterns. These examples illustrate how an application's structure can be organized to improve code quality, flexibility, and maintainability.

## Files Included

| File | Description |
|------|-------------|
| `02_MVC.py` | Demonstrates the Model-View-Controller (MVC) Pattern |
| `03_Dependency_Injection.py` | Demonstrates the Dependency Injection (DI) Pattern |

## Technologies Used

- Python 3.x
- Visual Studio Code
