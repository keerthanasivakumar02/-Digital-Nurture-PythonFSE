# Structural Design Patterns

## Description

This folder contains simple Python programs that demonstrate the implementation of Structural Design Patterns. These examples show how objects and classes can be connected to build flexible and maintainable software systems.

## Files Included

| File | Description |
|------|-------------|
| `02_Adapter.py` | Demonstrates the Adapter Pattern |
| `03_Decorator.py` | Demonstrates the Decorator Pattern |
| `04_Proxy.py` | Demonstrates the Proxy Pattern |

## Technologies Used

- Python 3.x
- Visual Studio Code
