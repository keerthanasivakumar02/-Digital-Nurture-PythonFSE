# Structural Design Patterns

## Introduction

Structural Design Patterns focus on organizing classes and objects to form larger and more efficient software structures. These patterns help different components work together smoothly while keeping the code simple, reusable, and easy to maintain.

## Design Patterns Included

- Adapter Pattern
- Decorator Pattern
- Proxy Pattern

## Advantages

- Makes object collaboration easier
- Improves code flexibility
- Encourages code reuse
- Reduces dependencies between classes
- Enhances the overall structure of an application
