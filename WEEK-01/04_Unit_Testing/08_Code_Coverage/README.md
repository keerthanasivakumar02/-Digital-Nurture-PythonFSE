# Code Coverage

## Description

This folder contains a simple example demonstrating how to measure code coverage using the `coverage.py` tool. It helps verify how much of the program is executed during testing and identifies any untested code.

## Files Included

- `calculator.py` – Contains basic arithmetic functions.
- `test_calculator.py` – Contains unit tests for the calculator functions.

## Technologies Used

- Python 3.x
- unittest
- coverage.py
- Visual Studio Code

## Installation

```bash
pip install coverage
```

## Running the Tests

```bash
python test_calculator.py
```

## Generating the Coverage Report

```bash
coverage run -m unittest test_calculator.py
coverage report
```
