# Jest

## Description

This folder contains a basic example of unit testing using the Jest framework. The project demonstrates how to test JavaScript functions using simple test cases.

## Files Included

- `calculator.js` – Contains basic arithmetic functions.
- `calculator.test.js` – Contains Jest test cases.
- `package.json` – Project configuration and Jest dependency.

## Technologies Used

- JavaScript
- Node.js
- Jest
- Visual Studio Code

## Installation

```bash
npm install
```

## Running the Tests

```bash
npm test
```
