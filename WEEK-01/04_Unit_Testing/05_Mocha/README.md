# Mocha

## Description

This folder contains a simple example demonstrating unit testing with the Mocha framework. It shows how to write and execute test cases for JavaScript functions in a Node.js environment.

## Files Included

- `calculator.js` – Contains basic arithmetic functions.
- `calculator.test.js` – Contains Mocha test cases.
- `package.json` – Project configuration and Mocha dependency.

## Technologies Used

- JavaScript
- Node.js
- Mocha
- Visual Studio Code

## Installation

```bash
npm install
```

## Running the Tests

```bash
npm test
```

or

```bash
npx mocha calculator.test.js
```
