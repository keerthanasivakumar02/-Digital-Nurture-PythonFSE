# Python unittest

## Description

This folder contains simple examples demonstrating unit testing using Python's built-in `unittest` framework. The examples show how to write test cases and verify the correctness of Python functions.

## Files Included

- `calculator.py` – Contains basic arithmetic functions.
- `test_calculator.py` – Contains unit test cases for the calculator functions.

## Technologies Used

- Python 3.x
- unittest
- Visual Studio Code

## Running the Tests

```bash
python test_calculator.py
```

or

```bash
python -m unittest test_calculator.py
```
