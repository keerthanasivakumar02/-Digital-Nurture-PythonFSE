# PyTest

## Description

This folder contains a basic example of using the PyTest framework to test Python functions. The examples demonstrate how to write test cases and verify that functions produce the expected results.

## Files Included

- `calculator.py` – Contains basic arithmetic functions.
- `test_calculator.py` – Contains PyTest test cases for the calculator functions.

## Installation

```bash
pip install pytest
```

## Running the Tests

```bash
pytest
```

## Technologies Used

- Python 3.x
- PyTest
- Visual Studio Code
