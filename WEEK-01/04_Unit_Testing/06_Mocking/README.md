# Mocking

## Description

This folder contains a simple example demonstrating the use of Python's `unittest.mock` library. Mock objects are used to simulate real functions, allowing unit tests to run independently of external dependencies.

## Files Included

- `service.py` – Contains a simple service function.
- `test_service.py` – Demonstrates mocking using `unittest.mock`.

## Technologies Used

- Python 3.x
- unittest
- unittest.mock
- Visual Studio Code

## Running the Program

```bash
python test_service.py
```

or

```bash
python -m unittest test_service.py
```
