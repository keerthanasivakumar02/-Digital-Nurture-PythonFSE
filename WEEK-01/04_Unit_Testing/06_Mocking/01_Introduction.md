# Mocking

## Introduction

Mocking is a software testing technique used to replace real objects or external dependencies with fake (mock) objects during testing. This allows developers to test individual functions without relying on databases, web services, or other external resources.

## Advantages

- Tests components independently
- Reduces testing time
- Eliminates dependency on external systems
- Improves test accuracy and reliability
- Makes unit testing easier

## Python Module Used

- `unittest.mock`
