# Test-Driven Development (TDD)

## Description

This folder contains a simple example demonstrating the Test-Driven Development (TDD) methodology using Python's built-in `unittest` framework. The example shows how tests can be written to verify program behavior before and during implementation.

## Files Included

- `calculator.py` – Contains a basic addition function.
- `test_calculator.py` – Contains unit tests for the calculator function.

## Technologies Used

- Python 3.x
- unittest
- Visual Studio Code

## Running the Tests

```bash
python test_calculator.py
```

or

```bash
python -m unittest test_calculator.py
```
