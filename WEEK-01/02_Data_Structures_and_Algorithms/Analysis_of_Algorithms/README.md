# Analysis of Algorithms

## Description

This folder contains simple examples that explain the basic concepts of algorithm analysis. The programs demonstrate how to evaluate algorithms based on execution time and memory usage using time complexity, space complexity, and asymptotic notation.

## Files Included

| File | Description |
|------|-------------|
| `02_Time_Complexity.py` | Demonstrates Time Complexity |
| `03_Space_Complexity.py` | Demonstrates Space Complexity |
| `04_Asymptotic_Notation.md` | Explains Big O, Big Ω, and Big Θ Notation |

## Technologies Used

- Python 3.x
- Visual Studio Code
