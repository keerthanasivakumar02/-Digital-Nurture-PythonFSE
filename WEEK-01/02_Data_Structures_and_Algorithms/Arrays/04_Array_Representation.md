# Array Representation

## Introduction

An array stores multiple elements of the same data type in continuous memory locations. Each element is identified by its position, called an index, which starts from **0**.

## Example

```text
Index :  0    1    2    3    4
Value : 10   20   30   40   50
```

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Access | O(1) |
| Search | O(n) |
| Insert | O(n) |
| Delete | O(n) |

## Summary

Arrays provide fast access to elements through indexing, but inserting or deleting elements may require shifting other elements, making those operations less efficient.
