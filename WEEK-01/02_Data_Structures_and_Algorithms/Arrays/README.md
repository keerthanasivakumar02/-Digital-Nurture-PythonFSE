# Arrays

## Description

This folder contains simple Python programs that demonstrate the basic operations performed on arrays. The examples help in understanding how arrays store data and how common operations such as traversal and searching are performed.

## Files Included

| File | Description |
|------|-------------|
| `02_Array_Traversal.py` | Demonstrates array traversal |
| `03_Array_Search.py` | Demonstrates searching for an element in an array |
| `04_Array_Representation.md` | Explains array representation and time complexity |

## Technologies Used

- Python 3.x
- Visual Studio Code
