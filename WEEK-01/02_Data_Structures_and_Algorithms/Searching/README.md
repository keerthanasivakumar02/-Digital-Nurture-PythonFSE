# Searching Algorithms

## Description

This folder contains simple Python programs demonstrating commonly used searching algorithms. These examples explain how to find an element in a collection using both linear and binary search techniques.

## Files Included

| File | Description |
|------|-------------|
| `02_Linear_Search.py` | Demonstrates the Linear Search algorithm |
| `03_Binary_Search.py` | Demonstrates the Binary Search algorithm |

## Technologies Used

- Python 3.x
- Visual Studio Code
