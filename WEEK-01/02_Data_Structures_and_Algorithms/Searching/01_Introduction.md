# Searching Algorithms

## Introduction

Searching algorithms are used to locate a specific element within a collection of data. Selecting the appropriate searching technique helps improve the speed and efficiency of applications, especially when working with large datasets.

## Algorithms Included

- Linear Search
- Binary Search

## Comparison

| Algorithm | Time Complexity | Requirement |
|-----------|-----------------|-------------|
| Linear Search | O(n) | Works with both sorted and unsorted data |
| Binary Search | O(log n) | Requires the data to be sorted |

## Applications

- Finding records in databases
- Searching files and folders
- Product search in e-commerce applications
- Student and employee record management
- Search engines
