# Sorting Algorithms

## Description

This folder contains simple Python programs demonstrating commonly used sorting algorithms. Each example shows how a collection of elements can be arranged in ascending order using different sorting techniques.

## Files Included

| File | Description |
|------|-------------|
| `02_Bubble_Sort.py` | Demonstrates Bubble Sort |
| `03_Insertion_Sort.py` | Demonstrates Insertion Sort |
| `04_Merge_Sort.py` | Demonstrates Merge Sort |
| `05_Quick_Sort.py` | Demonstrates Quick Sort |
| `06_Heap_Sort.py` | Demonstrates Heap Sort |

## Technologies Used

- Python 3.x
- Visual Studio Code
